# Support

This is the public support page for `not-manage`.

## Support channels

- Product questions, bug reports, and feature requests: GitHub issues at `https://github.com/Not-Operations/not-manage/issues`
- Private support, sales, or product contact: `hello@notoperations.com`
- Security reports: see `SECURITY.md`

## Before reaching out

Please include:

- your operating system
- your Node.js version
- the `not-manage` version
- the command you ran
- the exact error message
- whether the issue happened during setup, login, or a data command

Redact client data, tokens, app secrets, screenshots, and raw terminal output before sharing them.

## Common support topics

- setup and login: `README.md`
- privacy and runtime data path: `PRIVACY.md`
- sensitive data handling: `DATA-HANDLING.md`
- security reporting: `SECURITY.md`
- continuity and incident guidance: `OPERATIONS.md`

## Clio support boundary

Support for `not-manage` is provided by Not Operations, not by Clio.

If the issue is a Clio platform outage or API problem rather than a `not-manage` defect, contact Clio through their normal developer support channels.
